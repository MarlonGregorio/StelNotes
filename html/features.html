Where things are located

The website dropdown

The website name on the top left is a dropdown. In the dropdown are some navigation links to the home page, folders page, and public notebooks page. The user account page can also be found here alongside the sign-out or sign-in button depending on the user login status. At the end of the dropdown is the help link which links to this page.


The Navigation list

Right above this box is a list of links in the file explorer format. One goes to the home page and the other one represents the current page. This list will also appear on other pages with a different length, depending on how far someone is on the website.


The Back button

The Navigation list will disappear if a set of notes in a notebook is selected. The back button goes back to the main notebook. It will even have some of the notebook's title if not all to signal its purpose.


How to perform actions

Editing, deleting, and making a copy of notebooks/notes

After clicking on a notebook or a set of notes. There is a gear icon that acts as a dropdown. One can edit, delete, export, and import from the dropdown. If a notebook or a set of notes is going to be deleted, a warning prompt will be shown. If the notebook is in the "Shared Notebooks" folder it will be removed instead of deleted. Exporting will download a file of the notebook or set of notes data. Importing will restore the data of an exported file. Importing notes from Excel is also supported(xls or xlsx file) with 2 to 3 required columns. The first column is for terms, the second is for definitions, and the third(optional) is for examples or any extra info.


Sharing a notebook

Notebooks can be shared from the "My Notebooks" folder. Multiple notebooks can be shared at once to multiple recipients. No one has to be specified at the moment. Anyone with edit access can share with more people in the "Shared Notebooks" folder. When sharing, a permission(Edit or View) has to be selected. The owner of the notebook will always have edit access.


Resetting a password

If you forgot your password, you can request a reset from the login page on the last tab. The login page can be reached when "Get Started" is clicked on the homepage or anytime from the dropdown on the top left. If you still know your password and just want to update it, go to the account page from the website dropdown. You will be sent an email with a one time link by stelnotes@gmail.com.
